# Pages

Specific pages to process. Accepts a list of integers or a string of comma-separated numbers and ranges (e.g. '0,1,2' or '0-5' or '0,2-4'). Page numbers start from 0.


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number[]`

```typescript
const value: number[] = [
  339048,
  133621,
  943799,
];
```
