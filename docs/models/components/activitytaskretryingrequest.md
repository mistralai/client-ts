# ActivityTaskRetryingRequest

Emitted when an activity task fails and will be retried.

Contains information about the failed attempt and the error that occurred.

## Example Usage

```typescript
import { ActivityTaskRetryingRequest } from "@mistralai/mistralai/models/components";

let value: ActivityTaskRetryingRequest = {
  eventId: "<id>",
  rootWorkflowExecId: "<id>",
  workflowExecId: "<id>",
  workflowRunId: "<id>",
  workflowName: "<value>",
  attributes: {
    taskId: "<id>",
    activityName: "<value>",
    attempt: 660911,
    failure: {
      message: "<value>",
    },
  },
};
```
