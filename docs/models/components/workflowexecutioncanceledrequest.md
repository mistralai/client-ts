# WorkflowExecutionCanceledRequest

Emitted when a workflow execution is canceled.

This is a terminal event indicating the workflow was explicitly canceled.

## Example Usage

```typescript
import { WorkflowExecutionCanceledRequest } from "@mistralai/mistralai/models/components";

let value: WorkflowExecutionCanceledRequest = {
  eventId: "<id>",
  rootWorkflowExecId: "<id>",
  workflowExecId: "<id>",
  workflowRunId: "<id>",
  workflowName: "<value>",
  attributes: {
    taskId: "<id>",
  },
};
```
